# What's on the Menu?

A small browser quiz game for SFBBO where players match birds to the foods they eat.

## Local Development

Run the game with a simple static server from the repo root:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

To simulate the GitHub Pages project path locally:

```bash
cd ..
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/whats-on-the-menu/`.

## Tests
